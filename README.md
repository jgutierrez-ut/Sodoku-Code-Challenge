Sodoku-Code-Challenge
=====================

Java class to solve a Sodoku Puzzle 9*9 grid.
The program takes an unsolved Sodoku Puzzle in a comma separated file.
It will check for invalid inputs such as letter, extra columns or rows.
The ouput which is the solved Sodoku is place in a new file (sodoku_solved.csv) at the same location as the input file.
